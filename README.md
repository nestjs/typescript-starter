# clinestetic
App para segmento de estetica

# Pré-requisitos

Node.js, versão 10.13 ou superior;
Nest CLI (opcional, mas recomendado);
PostgreSQL, versão 9 ou superior (pode ser via Docker).
## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
